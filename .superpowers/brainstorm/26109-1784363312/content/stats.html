Server Stats & Visitor Locations

How should we track this?

Server Stats (Storage + RAM)

Simplest approach: a cron job on your server runs every minute and writes disk/memory usage to a JSON file. The PWA fetches it and shows a simple gauge or bar.

Example display:
Storage
342 GB / 500 GB
RAM
3.2 GB / 8 GB

Visitor Locations

Two ways to get this:

A. Free IP geolocation API

When someone visits the page, JS calls ip-api.com (free, no key) to get their city/country from their IP. Result is sent to a small logging endpoint on your server. Shows a list like "Dubai, AE • Paris, FR"

B. Cloudflare Analytics

If your server is behind Cloudflare, pull location data from their analytics API. Less setup but requires Cloudflare.

Click A or B to pick your preferred approach for location tracking.
