# Bitar Family PWA — Design Spec

## Overview

A Progressive Web App landing page for the Bitar family. Provides one-tap access to family-hosted VPS tools, helping the family transition away from Google services. Designed for mobile-first usage so family members can "install" it on their phone home screens.

## Tech Stack

- **Build tool:** Vite
- **PWA:** `vite-plugin-pwa` (auto-generates service worker + manifest injection)
- **Language:** Vanilla HTML/CSS/JS (no framework)
- **Icons:** Emoji for tool icons, brand-colored SVG/emoji for service comparisons

## Architecture

```
vpsLanding/
├── index.html              # Single page app shell
├── vite.config.js          # Vite config + PWA plugin setup
├── package.json            # Dependencies (vite, vite-plugin-pwa)
├── public/
│   └── icons/
│       ├── icon-192.png    # PWA icon 192x192
│       └── icon-512.png    # PWA icon 512x512
├── style.css               # Warm & homey theme styles
└── main.js                 # Builds tool cards, registers SW
```

**Note:** Service worker is auto-generated by `vite-plugin-pwa` at build time — no manual `sw.js` file needed.

**Data flow:**
- `main.js` defines a static array of tool objects (name, url, icon, description, similarTo)
- On load, it renders each tool as a card in a 2-column grid
- Tapping a card opens `url` in a new tab (`target="_blank"`)
- On load, fetches `/stats.json` (served from server) and renders storage/RAM gauges
- `vite-plugin-pwa` generates the service worker at build time, enabling offline caching and install prompt

**Server-side (outside the PWA build):**
- A cron job (e.g., every 5 min) runs `stats.sh` that writes disk & memory usage to `<webroot>/stats.json`
- Coolify serves the static files + the stats.json endpoint

## Visual Design

### Theme: Warm & Homey

- **Colors:** Warm neutrals (cream, tan, soft brown) with muted accents
- **Corners:** Rounded (12px cards, 10px icons)
- **Typography:** System font stack
- **Shadows:** Soft, minimal

### Layout (top to bottom)

1. **Top bar** — Centered `✦ Bitar Family` in a small pill/chip
2. **Tool grid** — 2-column card grid, each card:
   - Left: Emoji in a warm-toned rounded square
   - Name heading + one-line description below
   - Right: Familiar-service badge (e.g., "Google", "ChatGPT", "Drive") with brand-colored icon
3. **WireGuard section** — Below the tool grid. Card with:
   - 🌐 icon + "Secure Connection" heading
   - Explanation: "Install WireGuard and scan the QR code from Omar to access all tools securely."
   - Google Play badge/link button for the WireGuard app
4. **Server stats section** — Below the WireGuard section. Shows two gauges side-by-side:
   - Storage: used GB / total GB with color-coded progress bar (green → orange → red)
   - RAM: used GB / total GB with same progress bar treatment
   - Stats are fetched on page load from a static JSON endpoint (e.g., `/stats.json`) generated by a cron job on the server
5. **Footer** — "Built with ❤️ for the Bitar family" in small muted text

### Tool Cards Data

| Tool | Icon | Description | Similar To |
|------|------|-------------|------------|
| Excalidraw | ✏️ | Whiteboard & diagrams | Miro |
| n8n | 🔄 | Automate tasks | Zapier / IFTTT |
| Nextcloud | ☁️ | File storage & sharing | Google Drive |
| Omni-tools | 🧰 | Utility tools | Google Tools |
| SearXNG | 🔍 | Private search | Google Search |
| OpenWebUI | 🤖 | AI chat assistant | ChatGPT |

Each card shows the familiar-service badge (brand icon + name) so non-technical family members instantly recognize the equivalent service.

## PWA Setup

- **Manifest:** App name "Bitar Family", standalone display, theme color matching the warm palette
- **Service worker:** Generated by `vite-plugin-pwa` with precaching of all static assets
- **Icons:** 192x192 and 512x512 PNG icons. We'll generate simple placeholder icons with a warm-colored background and "BF" monogram. The user can replace with a family photo later.
- **Offline:** Basic precache — app renders even without network for the cached assets

## Success Criteria

- Installs as a PWA on iOS (Safari share sheet → Add to Home Screen) and Android (Chrome install prompt)
- All 6 tool cards visible without scrolling on most phones (or minimal scroll)
- Tapping any card opens the corresponding VPS service in a new browser tab
- Page loads in under 2s on mobile connection
- Server stats (storage + RAM) shown as progress bars with color-coded status
- WireGuard connection info section visible below the tool grid
- Cron-job-generated `/stats.json` updates every 5 minutes
